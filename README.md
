# Michael Mutie's Portfolio

A modern, responsive portfolio website showcasing my projects and skills as a Full Stack Developer.

## Features

- Responsive design that works on all devices
- Dark/light mode toggle
- Projects showcase with filtering
- Skills visualization
- Contact form with database integration
- Profile image upload functionality

## Technologies Used

- HTML - Structure and semantic markup
- CSS - Styling and responsive design
- JavaScript - Interactive elements and functionality
- Database Integration - For storing contact form submissions

## Deployment

This portfolio is configured for deployment on GitHub Pages, meeting the assignment requirement for successful hosting.

### How to Deploy

1. Fork or clone this repository
2. Connect it to your GitHub account
3. Enable GitHub Pages in your repository settings
4. The site will be automatically built and deployed when you push to the main branch
